2.differentiation tag
tag ความเหมือน ความแตกต่าง
<hr> & <br>, ขึ้นบรรทัดใหม่เหมือนกัน hr จะมีเส้นขีดด้านล่าง ,br จะขึ้นบรรทัดใหม่แต่ไม่มีเส้นขีด
<ul> & <ol>, แสดงข้อมูลเป็นลิสต์ ul ลิสต์รายการ แบบไม่มีลำดับ โดยใช้สัญลักษณ์กำกับ (Unordered List หรือ Bullet List)
ol ลิสต์รายการ แบบมีลำดับ (Ordered List)
<select> & <checkbox> & <radio button>, เป็นการเลือกข้อมูล แต่ถ้าข้อมูลที่ต้องเลือกมีเยอะ ควรเลือกใช้ select
แต่ถ้าผู้ใช้สามารถเลือกได้มากกว่าหนึ่งอย่างให้ใช้ checkbox
ถ้าผู้ใช้เลือกได้แค่อย่างเดียวให้ใช้ radio button
<div> & <span>, เป็นการจัดกลุ่มข้อมูล div จะมีคุณสมบัติ Style Display:block อยู่ในตัว ทำให้ทุก Element ที่อยุ่ต่อจากแท็ก Div จะถูกดันไปอยู่บรรทัดใหม่โดยอัตโนมัติ
span จะเป็นการจัดกลุ่มข้อความเป็นก้อนเหมือน div แต่จะไม่ขึ้นบรรทัดใหม่ให้
<script> & <style>, เป็นtag html style ใช้กําหนดคุณลักษณะ เช่น สี ขนาด font ตำแหน่งการจัดวาง ให้กับ HTML element ต่างๆ โดยจะกำหนดในหน้าเว็บเพจนั้นๆ หรือสร้างเป็นไฟล์ style sheet แยกเป็นอีกไฟล์ แล้วเรียกใช้อีกที
ส่วน script เช่น javascript, vbscript เป็น function เพื่อใช้งาน หรือจะเขียนฟังก์ชั่นเก็บเป็นไฟล์ .js ก่อนแล้วค่อยเรียกใช้งานก็ได้
